# 토스 Frontend Developer 면접 과제 🔥

## Getting started

```sh
yarn dev
```

section 분리

- 기능별로 분리한다? -> 계산기는 사용자의 입력값을 받는다.
- 아래 탭 기능들은 그럼 입력값을 받는가? -> 아니오 -> 입력값을 받지 않는 리스트형태이니 하나의 리스트 도메인으로 묶자.
- 리스트 도메인으로 묶고 난 이후에는? -> 묶었는데, 내부에 리스트를 보여주는 영역과 결과값을 보여주는 영역으로 나뉘어진다.
- 그러면 리스트 도메인은 어떻게? -> 계산기와는 다른 성질을 가졌으므로 탭으로 컨트롤한다는 의미에서 컨트롤러 내부에 두 가지로 나누자.
- 두개의 컴포넌트가 각각의 영역에서 일하자!
