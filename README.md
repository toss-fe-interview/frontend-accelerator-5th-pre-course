## Getting started

```sh
yarn dev
```

# 작업관련

1. 필요한 컴포넌트

- SavingsCalculatorPage => 최상단 페이지 컴
- ProductList => 입력에 맞게 필터링 되어서 렌더링 + 적금 선택가능
- CalculationResult => 입력 데이터랑 적금 상품 데이터로 계산하고, 추천상품 목록 렌더링

1. 필요한 데이터

- 클라이언트
  - 사용자가 입력한 데이터
  - 서버에서 받아온 데이터인데, 선택 유무 데이터추가된.
  - 탭 (적금 상품 | 계산 결과)
- 서버 : api call해서 받아온 데이터

# 순서

1. api call 테스트
2. 컴포넌트 나누기
3. 각 컴포넌트 개발

- SavingsCalculatorPage => 입력 폼이랑 탭데이터
- ProductList => 데이터 props로 받아서 입력된 정보로 필터링하기
- CalculationResult => 데이터 props로 받고, 계산 표츌
